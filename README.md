# 봇 주요 작동 방식
- Embed
- Message Component
- 슬래시 명령어 [ / ]
- 접두사 명령어 [ ! or = ] (일부 기능에서만 사용)

* 기본 모드 : node index.js
* 점검 모드 : node system-maintenance.js
